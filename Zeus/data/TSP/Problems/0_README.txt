Formatting of MasterFile
  FilesNames are in column A
  FileTypes are in column B

Formatting of DataFiles
  Dimension (number of cusomters) is in cell B4
  If standard TSP File (type 0) coordinates begin on row 7
  If weighted matrix TSP File (Type 1,2,3,4)
     Matrix weight begins on row 8
     If the file has coordinates they are 1 row after the matrix
